# 🌐 Project 01 - CV Website

Welcome to **Project01_CV_Website**!

This repository contains my first **Personal CV Website**, developed using **HTML5** and **CSS3**. The project is part of my Web Development learning journey, where I practice building responsive and well-structured websites while improving my frontend development skills.

---

# 📖 Project Overview

The goal of this project is to create a clean and professional online CV that showcases personal information, education, skills, and contact details.

This project helped me strengthen my understanding of HTML structure, CSS styling, layout design, and responsive web development.

---

# ✨ Features

- Responsive Layout
- Clean and Modern UI
- Personal Profile Section
- About Me Section
- Education Information
- Skills Section
- Contact Information
- Easy-to-Read Design
