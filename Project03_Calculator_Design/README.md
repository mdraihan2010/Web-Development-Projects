# 🧮 Project 03 - Calculator Design

Welcome to **Project03_Calculator_Design**!

This repository contains my **Calculator Design Project**, developed using **HTML5** and **CSS3**. The project is part of my Web Development learning journey, where I practice creating modern, responsive, and visually appealing user interfaces while improving my frontend development skills.

---

# 📖 Project Overview

The goal of this project is to design a clean and responsive calculator interface using HTML and CSS.

This project helped me strengthen my understanding of CSS Grid, Flexbox, responsive layouts, button styling, and modern UI design principles.
