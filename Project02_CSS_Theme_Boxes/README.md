# 🎨 Project 02 - CSS Theme Boxes

Welcome to **Project02_CSS_Theme_Boxes**!

This project demonstrates how to create customizable theme boxes using **CSS Variables** and control element stacking with the **z-index** property. It is part of my Web Development learning journey and focuses on building reusable, maintainable, and visually organized UI components.

---

# 📖 Project Overview

The objective of this project is to understand how CSS Variables simplify theme management and how the `z-index` property controls the stacking order of overlapping elements.

By combining these concepts, I created customizable theme boxes with different color schemes and layered layouts.
